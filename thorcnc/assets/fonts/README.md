# Bundled Fonts

Diese Schriften werden beim App-Start in die Qt-Font-Datenbank geladen, damit ThorCNC distro-unabhängig identisch aussieht.

| Font | Datei | Lizenz |
|------|-------|--------|
| Bebas Neue | `BebasNeue-Regular.ttf` | SIL OFL 1.1 (`OFL.txt`) |
| Oswald | `Oswald-Regular.ttf` | SIL OFL 1.1 (`OFL.txt`) |
| Barlow Condensed | `BarlowCondensed-Regular.ttf` | SIL OFL 1.1 (`OFL.txt`) |
| Roboto Condensed | `RobotoCondensed-Regular.ttf` | Apache 2.0 (`RobotoCondensed-LICENSE.txt`) |

Quellen: [Google Fonts](https://fonts.google.com/) via [fontsource](https://github.com/fontsource/fontsource).
